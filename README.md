<h4 align="center">配置说明</h4>
</br>
* 参考 Vue-Cli3 的框架配置格式，并将public文件夹移入src文件夹中
* 使用babel
* 包含以下Vue套件
** Vue2
** Vuex
** VueRouter

<h4 align="center">起步</h4>
</br>
    npm clone https://github.com/HTandMY/webpack-vue-start.git
    cd webpack-vue-start
    npm install
    npm run serve
